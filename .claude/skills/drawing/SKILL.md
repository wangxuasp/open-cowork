# 图纸数据集操作技能

本技能通过调用远程 API，完成 Teamcenter 内图纸数据集的查看等操作。

## API 基础信息

- **GET 请求**: `{TEAMCENTER_RICH_CLIENT_MICROSERVICE_URL}?[&其他参数]`
- **BASE_URL**: 由用户提供，或从上下文中获取httpGet, analyzeLocalFile工具已经定义

| 编号 | 操作     | 方法 | 说明                  |
| ---- | -------- | ---- | --------------------- |
| 1    | 分析图纸 | GET  | 分析指定uid的图纸内容 |

## 1：分析图纸

**用途**：分析指定文件绝对完整路径中的内容。{filePath}{message}参数根据实际情况替换为文件路径和消息内容。

**请求方式**：GET

### 使用流程

1. 使用`httpGet`工具把uid转换为本地文件完整路径。{BASE_URL}由工具决定，{uid}由用户提供或从上下文获取。

**请求地址**：`{BASE_URL}/dataset?action=export&uid={uid}`

2. 把第一步文件完整路径作为参数传和用户提出的问题递给`analyzeLocalFile`工具。{BASE_URL}由工具决定，{filePath}由第一步获取，{message}由用户提供。

**请求地址**：`{BASE_URL}/vl/file?&filePath={filePath}&message={message}`

**参数说明**：

| 参数       | 必填 | 说明               |
| ---------- | ---- | ------------------ |
| `filePath` | 是   | 文件本地的完整路径 |
| `message`  | 是   | 消息内容           |

---

## 通用错误处理

| status | 含义       | 响应用户                                  |
| ------ | ---------- | ----------------------------------------- |
| 200    | 操作成功   | ✅ 按各操作的成功提示响应                 |
| 400    | 参数错误   | ❌ 请求参数有误，请检查消息或文件名格式   |
| 404    | 对象不存在 | ❌ 未找到指定文件，请确认文件名称是否正确 |
| 500    | 服务器错误 | ❌ 服务器异常，请稍后重试                 |
| error  | 网络异常   | ❌ 无法连接服务器，请确认服务是否启动     |

---

## 参数缺失时的处理

| 缺失参数   | 操作 | 处理方式                     |
| ---------- | ---- | ---------------------------- |
| `BASE_URL` | 所有 | 询问用户 API 地址            |
| `filePath` | 所有 | 询问：「请提供文件完整路径」 |

---

## 安全要求

- 所有 UID、文件名只能来自用户明确提供的值，不得推断或伪造
- 删除操作必须经过用户二次确认，严禁自动执行
- 禁止将其他用户输入拼接到请求路径或请求体中
