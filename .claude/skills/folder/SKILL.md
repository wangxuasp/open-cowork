# 文件夹操作技能

## 功能说明

本技能通过向指定 API 发送 GET 请求，完成文件夹的创建、删除、查询等操作。

## API 基础信息

- **请求方式**: HTTP GET
- **URL 格式**: `{TEAMCENTER_RICH_CLIENT_MICROSERVICE_URL}?action={action}&uid={uid}`
- **BASE_URL**: 由用户提供，或从上下文中获取

## 前置条件

- 获得文件夹的 UID（格式示例：`gXhVErqOZ_jnSD`）
- UID 必须符合标准 UUID 格式，否则拒绝执行并重新询问

## 支持的操作

### 1. 创建文件夹（create）

**用途**: 创建一个新文件夹。使用 `httpGet` 工具，传入文件夹名称（将 {folderName} 替换为实际值）, 父级文件夹uid（将 {uid} 替换为实际值）：

**请求格式**:

```
GET {BASE_URL}?action=create&name={folderName}&uid={uid}
```

**参数说明**:
| 参数 | 必填 | 说明 |
|------|------|------|
| action | 是 | 固定值：`create` |
| name | 是 | 文件夹名称 |
| uid | 否 | 父级文件夹uid |

---

### 2. 删除文件夹（delete）

**用途**: 删除指定文件夹。使用 `httpGet` 工具，传入文件夹uid（将 {uid} 替换为实际值）：

**请求格式**:

```
GET {BASE_URL}?action=delete&uid={uid}
```

**参数说明**:
| 参数 | 必填 | 说明 |
|------|------|------|
| action | 是 | 固定值：`delete` |
| uid | 是 | 要删除的文件夹uid |

**注意**: 删除操作不可逆，执行前应向用户确认。

---

### 3. 查询文件夹（query）

**用途**: 查询文件夹详情。使用 `httpGet` 工具，传入文件夹uid（将 {uid} 替换为实际值）：

**请求格式（查询列表）**:

```
GET {BASE_URL}?action=query&uid={uid}
```

**参数说明**:
| 参数 | 必填 | 说明 |
|------|------|------|
| action | 是 | 固定值：`query` |
| uid | 是 | 指定文件夹uid |

# 使用流程

1. **确认参数**: 从用户输入或上下文中提取 `uid`、`BASE_URL`，以及操作类型所需的额外参数（如 `name`）
2. **构建 URL**: 按照对应操作的格式拼接完整请求 URL
3. **发送请求**: 使用 `httpGet` 工具发送 GET 请求
4. **处理响应**: 解析并展示响应结果

# 错误处理

- 若 `uid` 缺失，提示用户提供
- 若 `delete` 操作未提供 `uid`，要求用户确认要删除的文件夹 ID
- 若 API 返回错误码，展示错误信息并建议用户检查参数

## 参数缺失时的处理

| 缺失参数         | 处理方式                     |
| ---------------- | ---------------------------- |
| `BASE_URL`       | 询问用户 API 地址            |
| `name`（创建时） | 询问文件夹名称，或使用默认名 |
| `uid`（删除时）  | 必须询问，不得假设           |
